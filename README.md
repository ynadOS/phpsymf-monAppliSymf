# monAppliSymf

## Description
Ce projet est une application web construite avec le framework Symfony.

## Prérequis
- PHP >= 8.4.1
- Composer
- Symfony CLI (facultatif)
- Serveur web (Apache, Nginx, ou Symfony web server)
- Démarrage du server local : symfony serve:start
